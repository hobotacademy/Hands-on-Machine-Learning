## 💡 The Dual-Implementation Philosophy
What sets this repository apart is our **"Dual-Implementation"** approach. We believe that to truly master Machine Learning, one must understand both the *engine* and the *driver's seat*:

1.  **From Scratch (The Engine):** We derive and implement core algorithms from the ground up using only `NumPy` and `Math`. This ensures a deep understanding of the underlying calculus, linear algebra, and optimization logic.
2.  **Scikit-Learn (The Driver's Seat):** We implement the same algorithms using industry-standard libraries like `Scikit-Learn`. This focuses on building efficient pipelines, hyperparameter tuning, and professional model evaluation.

## 🛠️ Curriculum Structure

The repository is organized into progressive modules:

### Phase 1: Foundations & Exploratory Analysis
* **S01 - S02:** Python for AI Review

* **S05:** Data Visualization (Matplotlib & Seaborn)

### Phase 4: Supervised Learning - Classification
